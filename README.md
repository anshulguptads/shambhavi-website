# Shambhavi Innovation Lab

AI-native software for how India learns, works, and builds.

**Live site:** https://shambhavilabs.com

This is the source code for the Shambhavi Innovation Lab marketing website. Static HTML/CSS/JS, deployed via Cloudflare Pages.

## Structure

- `index.html` — homepage
- `about.html` — about the founders
- `aspirems.html`, `neurolink.html`, `sanadeep.html`, `insighttest.html` — product pages
- `blog/` — journal posts
- `og-*.png` — Open Graph preview images
- `sitemap.xml`, `robots.txt` — SEO

## Deployment

Auto-deploys to Cloudflare Pages on push to `main`. No build step.
